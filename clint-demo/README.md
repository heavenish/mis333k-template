**Quick Setup**

**Clone the repo**:
git clone repo_clone_url

**Start the backend**: 
Navigate to the backend folder and run:
cd clint-demo/backend
node server.js

**Start the frontend**: 
Navigate back to the main project folder and run:
cd clint-demo
npm install
npm start

Access the app:
Frontend: http://localhost:3000
Backend: http://localhost:5000
